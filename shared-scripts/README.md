# 共享脚本库

此目录包含可在不同CI/CD平台和项目中复用的通用脚本。

## 脚本列表

1. `config-manager.sh` - 配置管理库
2. `doc-generator.sh` - 文档生成器
3. `logging.sh` - 统一日志记录库
4. `utils.sh` - 通用工具函数库
5. `validation.sh` - 参数验证库

## 使用方法

1. 在CI/CD流水线中引用相应脚本
2. 根据需要调整脚本中的函数和配置
3. 遵循脚本中的使用说明和参数要求

## 设计原则

- **模块化**: 每个脚本职责单一，可独立使用
- **可复用**: 脚本设计通用，适用于多种场景
- **易集成**: 提供清晰的接口和文档
- **可扩展**: 支持自定义配置和扩展
