## 🚀 Quick Start

### 1. Clone the Project
```bash
git clone https://github.com/Baozhi888/CICD-solution.git
cd CICD-solution
```

### 2. Set Permissions
```bash
chmod +x scripts/*.sh
chmod +x tests/*.sh
```

### 3. Run Tests
```bash
./tests/run-tests.sh
```
